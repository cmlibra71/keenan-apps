# Keenan Apps

Apps in this monorepo auto-deploy to the Keenan Portal sandbox on push.
See `CLAUDE.md` for the agent context (SDK reference, design system, deploy flow).

## Quick start

1. Create a folder named after your app slug (lowercase, hyphenated)
2. Add an `index.html` (required) and any other files
3. In the Command Center, create an app and set its **GitHub Source** to your folder name
4. Push to `main` — the app deploys within ~30 seconds
